Reading guide

What these analytics are telling you

Performance ladder

Ranks bots by current return, so you can see whether one runtime is carrying the fleet or multiple bots are performing consistently.

Health matrix

Puts success rate, action volume, error counts, and heartbeat freshness in one line per bot. A noisy runtime becomes obvious fast.

Exposure concentration

Shows where current risk is clustering by symbol. If several bots are leaning into the same market, you can see that concentration immediately.
